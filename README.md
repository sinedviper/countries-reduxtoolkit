# Countries-Client

### :pushpin: About project
<p align="justify">The project was written to learn redux thunk, react, axios and proper interaction with the store. As an example, I took the issuance of countries by search.</p>

---

### :book: Libraries

- react
- react-dom
- react-select
- react-icons
- axios
- redux
- react-redux
- redux-thunk
- @redux/toolkit
- styled-component

---

### :pizza: Commands

- npm run start: <strong>run in development mode</strong>
- npm run build: <strong>assemble the project</strong>
